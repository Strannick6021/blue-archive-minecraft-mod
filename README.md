# Blue Archive Minecraft Mod

## Описание

Мод добавляет в Minecraft систему компаньонов с поддержкой кастомных персонажей, оружием (дробовик, щит) и полноценным AI для добычи ресурсов, защиты и боевых действий.

## Возможности

### 1. Система персонажей
- Трансформация в кастомного персонажа
-召唤 компаньона для помощи
- Диалоги на русском языке

### 2. Оружие и предметы
- **Дробовик** - стреляет дробью с разбросом, требует перезарядки
- **Щит** - поглощает 70% урона в радиусе активации

### 3. Компаньон AI
Компаньон может:
- **Следовать** за игроком
- **Добывать ресурсы** - ищет блоки и добывает их автоматически
- **Защищать** - атакует враждебные мобы и защищает от врагов
- Плавать, карабкаться и ориентироваться в мире

## Установка

### Требования
- Java 21+
- Forge 1.21-51.0.0
- GeckoLib 4.1.0

### Шаги установки

1. **Скачай репозиторий:**
   ```bash
   git clone https://github.com/Strannick6021/blue-archive-minecraft-mod.git
   cd blue-archive-minecraft-mod
   ```

2. **Собери мод:**
   ```bash
   ./gradlew build
   ```
   На Windows:
   ```bash
   gradlew.bat build
   ```

3. **Готовый JAR файл:**
   Появится в папке `build/libs/BlueArchiveCompanion-1.0.0.jar`

4. **Установка:**
   - Скопируй JAR в папку `.minecraft/mods`
   - Убедись, что установлен Forge и GeckoLib
   - Запусти Minecraft

## Команды

```
/summon companion ~ ~ ~          - Призвать компаньона
/companion follow               - Компаньон следует за тобой
/companion defense              - Компаньон защищает от врагов
/companion mining <block_type>  - Компаньон добывает блоки
```

## Структура проекта

```
src/main/java/com/strannick/companion/
├── CompanionMod.java           # Главный класс мода
├── registry/                   # Реестры предметов, сущностей, звуков
├── item/                       # Предметы (дробовик, щит)
├── entity/                     # Сущности (компаньон, снаряд��)
└── companion/                  # AI система компаньона
```

## Оптимизация

- AI обновляется каждые 20 тиков (1 сек) вместо каждого тика
- Снаряды дробовика имеют ограниченное время жизни
- Поиск врагов в ограниченном радиусе
- NBT теги для минимальной синхронизации

## Безопасность

✅ Валидация на сервере для всех действий
✅ Защита от дюп-багов при расходе боеприпасов
✅ Проверка прав владельца компаньона
✅ Синхронизация клиент-сервер
