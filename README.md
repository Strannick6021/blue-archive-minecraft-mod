# Blue Archive Minecraft Mod

## Описание

Мод добавляет в Minecraft систему компаньонов с поддержкой кастомных персонажей, оружием (дробовик, щит) и полноценным AI для добычи ресурсов, защиты и боевых действий.

## Возможности

### 1. Система персонажей
- Трансформация в кастомного персонажа
-召唤 компаньона для помощи
- Диалоги на русском языке

### 2. Оружие и предметы
- **Дробовик** - стреляет дробью с разбросом, требует перезарядки
- **Щит** - поглощает 70% урона в радиусе активации

### 3. Компаньон AI
Компаньон может:
- **Следовать** за игроком
- **Добывать ресурсы** - ищет блоки и добывает их автоматически
- **Защищать** - атакует враждебные мобы и защищает от врагов
- Плавать, карабкаться и ориентироваться в мире

## Установка

### 1. Скачай репозиторий:
```bash
git clone https://github.com/Strannick6021/blue-archive-minecraft-mod.git
cd blue-archive-minecraft-mod
```

### 2. Собери мод:
```bash
./gradlew build
```
На Windows:
```bash
gradlew.bat build
```

Первая сборка займёт **3-5 минут** (Gradle качает зависимости).

### 3. Готовый JAR файл:
Появится в папке `build/libs/BlueArchiveCompanion-1.0.0.jar`

### 4. Установка:
- Скопируй JAR в папку `.minecraft/mods`
- Убедись, что установлены:
  - **Forge** для 1.21
  - **GeckoLib** (скачай отдельно)
- Запусти Minecraft

## Команды

```
/summon companion ~ ~ ~          - Призвать компаньона
/companion follow               - Компаньон следует за тобой
/companion defend               - Компаньон защищает от врагов
/companion mining <block_type>  - Компаньон добывает блоки
```

## Куда добавлять модели персонажей

### Структура папок для модели:
```
src/main/resources/assets/companion/
├── models/
│   ├── item/
│   │   ├── shotgun.json
│   │   └── shield.json
│   └── entity/
│       └── companion/
│           ├── companion.geo.json      (GeckoLib модель)
│           └── companion.animation.json
├── textures/
│   ├── item/
│   │   ├── shotgun.png (64x64 рекомендуется)
│   │   └── shield.png (64x64 рекомендуется)
│   └── entity/
│       └── companion/
│           └── companion.png (256x256 рекомендуется для персонажа)
└── sounds/
    ├── shotgun_fire.ogg
    ├── shotgun_reload.ogg
    ├── companion_affirm.ogg
    ├── companion_deny.ogg
    ├── companion_attack.ogg
    └── companion_mining.ogg
```

### Как создать модель в Blockbench:

1. Скачай **Blockbench** (https://blockbench.net/)
2. Создай новый проект `Modded Entity` для версии Forge 1.20+
3. Смоделируй персонажа (ноги, туловище, руки, голова, аксессуары)
4. Добавь текстуру (PNG 256x256 или больше)
5. Экспортируй как `.geo.json` (для GeckoLib)
6. Создай анимации (ходьба, стояние, атака) в Blockbench
7. Экспортируй анимации как `.animation.json`
8. Положи файлы в папки выше

### Как добавить текстуры:

1. Сохрани текстуру персонажа как `companion.png` (256x256 пикселей)
2. Положи в папку `textures/entity/companion/`
3. Убедись, что имя совпадает с имером в `.geo.json` модели

## Оптимизация

- ✅ AI обновляется каждые 20 тиков (1 сек) вместо каждого тика
- ✅ Поиск ресурсов кешируется на 100 тиков (5 сек)
- ✅ Поиск врагов кешируется на 50 тиков (2.5 сек)
- ✅ Снаряды дробовика имеют ограниченное время жизни
- ✅ Использование NBT для минимальной синхронизации

## Безопасность

✅ Валидация на сервере для всех действий
✅ Защита от дюп-багов при расходе боеприпасов
✅ Проверка прав владельца компаньона
✅ Синхронизация клиент-сервер через NBT
